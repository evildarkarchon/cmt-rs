### `src/main.rs` (route, request-response)

**Analog:** existing console entry point + Slint startup research.

**Current entry point to replace** (`src/main.rs` lines 1-3):

```rust

}
```

**Rust startup pattern** (`01-RESEARCH.md` lines 297-303):

```rust
// Source: https://docs.slint.dev/latest/docs/rust/slint/index.html
```

**Apply:** replace the `println!` bootstrap with generated Slint module inclusion and `MainWindow::new()?.run()`. Keep startup thin; do not embed tab behavior or domain logic in `main.rs`.

### `ui/main.slint` (component, event-driven)

**Analog:** reference Tk shell/tab construction + Slint `TabWidget` standard pattern.

**Reference window/tab construction** (`CMT/src/cm_checker.py` lines 79-104):

```python
def setup_window(self) -> None:
	self.root.wm_resizable(width=False, height=False)
	self.root.wm_attributes("-fullscreen", "false")
	self.root.wm_iconphoto(True, self.get_image("images/icon-32.png"))
	self.root.wm_title(f"{APP_TITLE} v{APP_VERSION}")
	self.root.wm_protocol("WM_DELETE_WINDOW", self.on_close)

	notebook = ttk.Notebook(self.root)
	notebook.grid(column=0, row=1, sticky=NSEW)

	self.tabs: dict[Tab, CMCTabFrame] = {
		Tab.Overview: tabs.OverviewTab(self, notebook),
		Tab.F4SE: tabs.F4SETab(self, notebook),
		Tab.Scanner: tabs.ScannerTab(self, notebook),
		Tab.Tools: tabs.ToolsTab(self, notebook),
		Tab.Settings: tabs.SettingsTab(self, notebook),
		Tab.About: tabs.AboutTab(self, notebook),
	}

	notebook.bind("<<NotebookTabChanged>>", self.on_tab_changed)
```
