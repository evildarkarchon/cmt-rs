## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Text/icon gaps if any inline decoration is later needed |
| sm | 8px | Tab placeholder internal gaps and compact padding |
| md | 16px | Default placeholder block padding inside each tab |
| lg | 24px | Main tab content inset from tab edges |
| xl | 32px | Major vertical separation if a placeholder needs heading plus body |
| 2xl | 48px | Reserved for future full tab layouts; avoid in Phase 01 placeholders |
| 3xl | 64px | Reserved for future page-level layouts; avoid in Phase 01 placeholders |

Exceptions: none. Phase 01 has no touch-specific controls; tab hit targets should use Slint `TabWidget` defaults.

---

## Typography

Use exactly these roles and no additional text scale in Phase 01.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label | 14px | 400 | 1.5 |
| Heading | 18px | 600 | 1.2 |
| Display | 20px | 600 | 1.2 |

Typography constraints:
- Tab titles must use Slint `TabWidget` defaults so the shell stays close to the platform-native Tk notebook feel.
- Placeholder headings may use Heading; placeholder bodies must use Body.
- Do not use bold except the 600 Heading/Display weights above.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | platform window background / `#f0f0f0` fallback | Window background and tab content surface |
| Secondary (30%) | platform control surface / `#ffffff` fallback | Placeholder panels if a tab component needs a bounded content surface |
| Accent (10%) | platform focus/accent / `#2563eb` fallback | Selected tab indicator/focus affordance only when provided by Slint defaults |
| Destructive | `#b91c1c` | Not used in Phase 01; destructive actions are out of scope |

Accent reserved for: selected tab/focus affordance only. Do not use accent for placeholder text, fake action links, or decorative emphasis in Phase 01.
